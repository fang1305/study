	vue(渐进式JavaScript框架)基本api要会
(1)前端框架与库的区别
    框架=全方位功能齐全( 简易的DOM体验+发请求+模板引擎+路由功能 )
    举例: 
        KFC世界中: 库是小套餐, 框架是全家桶
    代码的不同: 
        一般使用库的代码,是调用某个函数,我们自己把控库的代码
        一般使用框架,其框架在帮我们运行我们编写好的代码
            框架: 初始化自身的一些行为
                执行你所编写的代码
                施放一些资源
(2) 起步
    MVVM
        Model View ViewModel

    1. 引包
    2. 启动new Vue (el: 目的地 template: 模板内容)
    options: 
        目的地: el
        内容: template
        数据: data 保存数据属性(在组件中,data必须是一个函数)
